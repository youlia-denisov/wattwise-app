# ⚡ WattWise — Multi-User Electricity Dashboard

A multi-user Streamlit web app for analyzing household electricity consumption and finding the best electricity discount plan in the Israeli market. Each user uploads their own CSV file and runs the full analysis pipeline independently — no data leaks between sessions.

Built as a modular refactor of the [Electricity Consumption Analyser](../electricity-consumption-analyser) project.

---

## Quick Start

### 1. Clone & setup

```bash
git clone https://github.com/YouliaXX/DS-portfolio.git
cd DS-portfolio/streamlit-multiuser-app

python -m venv venv
# Windows:
venv\Scripts\activate
# macOS / Linux:
source venv/bin/activate
```

### 2. Install dependencies

```bash
pip install -e .
```

### 3. Run the app

```bash
streamlit run streamlit_electricity_usage.py
```

### 4. Use the app

1. Upload your IEC electricity CSV in the sidebar
2. Set your tariff and smart-meter status
3. Click **Run Full Pipeline**
4. Explore the tabs

## Features

- **Multi-user sessions** — each browser session gets an isolated temp directory; uploads and results never mix between users
- **Overview** — high-level consumption summary and cost estimate
- **Hourly Patterns** — time-of-day usage breakdown
- **Clustering** — k-means grouping of hourly readings into 4 ranked usage profiles
- **Behavioural Fingerprint** — compact metrics describing your household's usage habits (WFH ratio, peak hour, regularity, etc.)
- **Outlier Detection** — side-by-side comparison of 4 methods: 3-sigma, IQR, DBSCAN, Isolation Forest — with auto-selected recommendation
- **Discount Calculator** — ranks available Israeli electricity plans by estimated savings on your actual data

## Input Data

### Your electricity file
Expects a CSV export from the **Israeli Electric Corporation (IEC)** digital meter portal. The app handles Hebrew headers, metadata rows, and European decimal separators automatically.

How to get it:
1. Log in to the [IEC self-service portal](https://www.iec.co.il)
2. Navigate to **My Consumption → Export data**
3. Download the CSV and upload it directly in the app sidebar — no manual setup needed

## Discount Offers Data

The Discount Calculator tab depends on `data/external/electricity_discount_offers.csv`, which contains current electricity discount plans from Israeli providers (Cellcom, Hot, Bezeq, Partner, and others).

**This file is included in the repository** so the app works immediately after cloning. However, discount offers change frequently — providers update their plans, add conditions, or remove deals. It is recommended to refresh the file every few weeks.

### Why it needs refreshing

Israeli electricity providers regularly update their discount percentages, time restrictions, and eligibility conditions. An outdated file means the calculator compares your usage against stale offers, which could lead to a wrong recommendation.

### How to update it

The file is generated by asking Claude (claude.ai) to scrape the provider pages. A ready-made prompt is included in the repo:

1. Open [`docs/kamaze_scraping_ai_prompt.md`](./docs/kamaze_scraping_ai_prompt.md)
2. Copy the entire contents
3. Go to [claude.ai](https://claude.ai) and select model **Claude Sonnet** (any recent version)
4. Paste the prompt and send it — Claude will scrape the provider pages and return a CSV
5. Save the result as `data/external/electricity_discount_offers.csv`, replacing the old file

> **Note:** Some provider pages are JavaScript-rendered and may not load in a plain browser fetch. If Claude reports `FAILED:JS_RENDERED` for a URL, the data for that provider could not be retrieved. The remaining providers will still be included. Try again later or on a different model if this happens frequently.

### What the file contains

| Column | Description |
|---|---|
| `supplier_name` | Provider name (e.g. Cellcom, Hot) |
| `plan_name` | Name of the specific plan |
| `discount_pct` | Discount percentage (1–30) |
| `discount_type` | `discount` (off bill) or `cashback` (wallet credit) |
| `requires_smart_meter` | Whether a smart meter is needed |
| `time_restriction` | Hours when the discount applies (if any) |
| `customer_type` | Who is eligible (e.g. existing subscribers) |
| `deal_url` | Direct link to the plan page |
| `scraped_at` | Date the data was collected |
